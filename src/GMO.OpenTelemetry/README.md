# GMO.OpenTelemetry

OpenTelemetry instrumentation for .NET 10+: runtime, process, and custom CPU % metrics.

**Install:** Add the package from [GitHub Packages](https://github.com/gideonogegaorg/OpenTelemetry/packages) (feed: `https://nuget.pkg.github.com/gideonogegaorg/index.json`). Authenticate with a PAT with `read:packages`.

**Full documentation:** [README](https://github.com/gideonogegaorg/OpenTelemetry/blob/main/README.md)
