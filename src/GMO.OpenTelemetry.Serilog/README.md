# GMO.OpenTelemetry.Serilog

Serilog integration for GMO.OpenTelemetry (.NET 10+). Depends on **GMO.OpenTelemetry**.

**Install:** Add the package from [GitHub Packages](https://github.com/gideonogegaorg/OpenTelemetry/packages) (feed: `https://nuget.pkg.github.com/gideonogegaorg/index.json`). Install **GMO.OpenTelemetry** first if needed.

**Full documentation:** [README](https://github.com/gideonogegaorg/OpenTelemetry/blob/main/README.md)
